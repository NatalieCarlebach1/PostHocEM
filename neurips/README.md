# NeurIPS 2025 — Post-hoc Entropy Minimization (PEM)

Full-length submission extending the MIDL 2026 short paper at `../paper/`.

## Relation to the MIDL paper

This subfolder holds the NeurIPS-specific LaTeX, experiment configs, figure-
generation scripts, and results. **It does not duplicate the MIDL code**:
Python modules (`networks/`, `utils/`, `dataloaders/`, training scripts like
`train_bcp_baseline.py`) live at the **repo root** and are imported from
`neurips/experiments/launch/*.py` as needed.

The MIDL 3-page paper at `../paper/main.tex` stays frozen on its own branch
(`feature/midl-3page-shrink`); NeurIPS work lives on
`feature/neurips-bootstrap` and beyond.

## Layout

```
neurips/
├── paper/              # NeurIPS LaTeX (main.tex + sections/)
│   ├── main.tex
│   ├── macros.tex
│   ├── references.bib
│   ├── sections/       # \input'd from main.tex
│   ├── figures/        # PDFs generated by neurips/figures/*.py
│   ├── tables/         # .tex auto-generated from neurips/results/
│   └── build/          # latexmk output (gitignored)
│
├── experiments/        # one YAML config + one launcher per run
│   ├── configs/
│   │   └── sweeps/
│   ├── launch/
│   └── README.md
│
├── figures/            # figure-generation scripts (read results/, write paper/figures/)
├── results/            # single source of truth for every number in the paper
├── tests/              # pytest for NeurIPS-specific additions
└── README.md
```

## Reproduce a result

```bash
# From repo root (not neurips/)
bash neurips/experiments/launch/run_single.sh neurips/experiments/configs/<exp>.yaml
```

## Build the paper

```bash
cd neurips/paper && latexmk -pdf main.tex
```

## NeurIPS style files

Drop `neurips_2025.sty` (from the current NeurIPS CFP page at `neurips.cc`) into
`neurips/paper/` and uncomment the `\usepackage{neurips_2025}` line in `main.tex`.
Not checked in; each year has its own.

## Experiment naming

```
<dataset>_<labelfrac>_<base>_<method>_<variant>_<seed>
```

Examples:
- `pancreas_20_bcp_pem_tau099_s2020`
- `la_5_bcp_plft_s42`
- `acdc_10_most_pem_full_s2020`
- `brats_5_dycon_pem_tau095_s42`

Every entry under `neurips/results/` uses this as its folder name. Every paper
table/figure traces back to one or more `<exp_id>`s via `config.yaml` frozen
inside that folder.
